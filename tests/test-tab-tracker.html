TabTracker Tests

Manual Test Procedures

These tests must be run in Firefox via about:debugging with the extension loaded:

1. Open 3 tabs to walmart.com
   → Check background console: count should show 3

2. Close 1 tab
   → Count decrements to 2

3. Close remaining 2 tabs
   → Count reaches 0, "Close timer started" logged
   → After 500ms, "Domain closed: walmart.com" logged

4. Open 3 tabs to walmart.com, close all, then within 500ms open a new walmart.com tab
   → "Close timer cancelled" logged, count = 1

5. Navigate a tab from walmart.com to target.com
   → walmart.com count decrements, target.com count increments

6. Open about:newtab
   → No tracking — restricted URL ignored

7. Restart the extension via about:debugging
   → "TabTracker initialized" log with correct tab/domain counts

8. Run in background console:
   TabTracker.getDomainForTab(tabId)  → returns domain for a known tab
   TabTracker.getTabCount("walmart.com")  → returns current count
